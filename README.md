# Syst-me-de-Gestion-de-Biblioth-que
Description :
Site web permettant l’inscription et la connexion des utilisateurs, l’ajout de livres, la recherche et la consultation des livres disponibles, ainsi que le suivi des emprunts avec mise à jour automatique de la disponibilité.

Technologies utilisées : PHP, MySQL, HTML, CSS, Sessions PHP

Objectifs / Résultats :

Création d’un site web complet intégrant authentification, gestion de contenu et suivi des états des livres.

Création d’une interface simple et intuitive pour la gestion d’une bibliothèque.
